# NanoFramework
Smallest PHP framework

## Instalação
Clone o repositório
```shell
git clone https://github.com/Eskelsen/NanoFramework.git .
```
Habilite o utilitário
```shell
chmod +x ./nano
```
Inicie o framework
```shell
./nano start
```
Caso não seja habilitável (ou Windows)
```shell
php nano start
```
Requer a função shell_exec habilitada
